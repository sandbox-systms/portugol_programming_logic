# 📘 Portugol Programming Logic

Repositório dedicado ao estudo de **lógica de programação utilizando Portugol (português estruturado)**, com foco em construção de base sólida para desenvolvimento de software.

---

## 📌 Objetivo

Este projeto tem como finalidade:

- Desenvolver raciocínio lógico aplicado à programação  
- Ensinar fundamentos essenciais antes de linguagens como C, Python ou Java  
- Fornecer exercícios progressivos e exemplos práticos  
- Servir como material de apoio para estudo autodidata ou ensino  

---

## 🧠 Conteúdos Abordados

Os algoritmos presentes neste repositório cobrem:

- Variáveis e tipos de dados  
- Entrada e saída (`leia`, `escreva`)  
- Operadores aritméticos e lógicos  
- Estruturas condicionais (`se`, `senao`)  
- Estruturas de repetição (`para`, `enquanto`)  
- Funções e modularização  
- Problemas matemáticos e do mundo real
