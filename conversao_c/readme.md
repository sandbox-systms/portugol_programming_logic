# Conversão de Portugol para C

Esta pasta contém programas de exemplo e exercícios que foram **convertidos de Portugol para a linguagem C**. Os arquivos demonstram conceitos fundamentais de programação como tipos de dados, estruturas condicionais (if-else) e operações com entrada e saída de dados.

## 📁 Estrutura dos Arquivos

### Exemplos de Tipos de Dados
- **`dataTypes.c`** — Demonstra o uso de diferentes tipos de dados (int, float, char) com um exemplo prático de compra em uma loja, mostrando como declarar, atribuir e exibir variáveis.

### Exercícios Práticos
- **`conditions.c`** — Exemplo simples de estrutura condicional if-else que compara um número com 10.
- **`if_else-E.c`** — Exercício com condicionais if-else.
- **`if-else-or.c`** — Exercício usando operadores lógicos (OR) em estruturas condicionais.
- **`exercises.c`** — Programa que calcula a **média de notas de um aluno** a partir de 4 notas digitadas.
- **`lanHouse.c`** — Sistema de cálculo de **cobrança em LAN House**, convertendo minutos utilizados em valor a pagar (R$ 2,00 por hora).

## 🎯 Objetivo da Pasta

Facilitar o aprendizado de programação em C mostrando como algoritmos escritos em **Portugol** (linguagem educacional em português) são traduzidos para **C** (linguagem mais profissional e amplamente usada).

## 🔧 Como Compilar e Executar

Para compilar um programa:
```bash
gcc nome_arquivo.c -o nome_saida
```

Para executar:
```bash
./nome_saida
```

**Exemplo:**
```bash
gcc exercises.c -o exercises
./exercises
```

## 📝 Conceitos Abordados

- Tipos de dados: `int`, `float`, `char`
- Entrada de dados: `scanf()`, `fgets()`
- Saída de dados: `printf()`
- Estruturas condicionais: `if-else`
- Operadores lógicos
- Operações matemáticas básicas

## 📦 Executáveis Pré-compilados

Os arquivos compilados estão disponíveis na pasta `../output/` para execução direta sem necessidade de compilação.
